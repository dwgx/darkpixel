# darkpixel

Minecraft Paper/Spigot server plugin helper.

## Status

Experimental plugin project. Confirm the target Minecraft/Paper version before installing it on a real server.

## Setup

Install a compatible JDK and the build tool used by this repository, then build the plugin jar locally.

## Usage

Copy the built jar to a test server first. Do not deploy directly to a production server without testing.

## Configuration

Keep server addresses, tokens, database passwords, and private player data out of the repository.
